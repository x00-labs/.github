# x00-labs/.github

This is the **community health and organization profile** repository for [x00-labs](https://github.com/x00-labs).

## What's in this repository

| Path | Purpose |
|------|---------|
| [`profile/README.md`](profile/README.md) | Public organization profile shown on the x00-labs GitHub page |

## Purpose

GitHub treats the `.github` repository in an organization as a special repository. Files placed here serve the entire organization:

- **`profile/README.md`** — Displayed on the organization's public GitHub profile page.
- **`CONTRIBUTING.md`** — Default contributing guide for all repositories in the org.
- **`CODE_OF_CONDUCT.md`** — Default code of conduct for all repositories in the org.
- **`SECURITY.md`** — Default security policy for all repositories in the org.
- **`ISSUE_TEMPLATE/`** — Default issue templates for repositories that don't define their own.
- **`PULL_REQUEST_TEMPLATE.md`** — Default pull request template.
- **`.github/workflows/`** — Reusable GitHub Actions workflows.
